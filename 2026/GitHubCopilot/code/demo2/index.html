GitHub Copilot: very simple examples

Start Copilot with copilot. Then use an agent for a type of task, a skill for a reusable instruction, and a hook to run a command automatically.

Agent

Use a built-in agent

Ask Copilot to use a specialist agent for the job.

/agent

Pick: code-review

Prompt:
Review my changes and tell me
only the real problems.
Custom agent

Create one tiny agent

Put this file in .github/agents/simple-html.agent.md.

---
name: simple-html
description: Help make very small HTML examples.
tools: read, edit
---

Keep examples tiny.
Prefer one file.
Skill

Create one tiny skill

Put this file in .github/skills/html-helper/SKILL.md.

---
name: html-helper
description: Help write very small HTML pages.
---

Keep the HTML simple.
Use one file when possible.
Avoid frameworks unless asked.
Use skill

Call the skill

Mention the skill in your prompt.

Use the /html-helper skill
to make a simple landing page.
Hook

Create one tiny hook

Put this file in .github/hooks/simple.json.

{
  "version": 1,
  "hooks": {
    "sessionStart": [
      {
        "type": "command",
        "powershell": "Write-Host 'Copilot session started'"
      }
    ]
  }
}
